## Contract

v0.24.0 defines one local read-only evidence foundation that can describe Binance Spot canary evidence and Binance USD-M Futures read-only evidence in the same OMS, Portfolio, Risk, Reconciliation, failure classification, Dashboard and CLI vocabulary.

`v0.23.1` must be complete before this foundation is executable.

## Boundary

This contract does not enable Futures order execution, OKX active runtime, production cutover, production secret read, production endpoint connection, broker endpoint connection, Dashboard trading button, order form, or live command.
