## Goal

v0.23.0 is Binance USD-M Futures read-only foundation. It introduces venue / product / environment identity, read-only endpoint allowlist, credential reference gate, redacted account snapshot shape, position / margin / leverage observation, funding / mark / liquidation-risk observation, transport failure classification, local registry reconciliation, and Dashboard / CLI read-only readiness surface.

## Non-goals

- No Futures order execution.
- No signed order endpoint.
- No submit / cancel / replace.
- No broker reconciliation runtime.
- No OKX active path.
- No production cutover.

## Boundary

`futuresOrderExecutionEnabled=false`, `productionTradingEnabledByDefault=false`, and production cutover not authorized.
