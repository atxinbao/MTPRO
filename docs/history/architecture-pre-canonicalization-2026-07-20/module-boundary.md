# Module Boundary

日期：2026-06-14

执行者：Codex

本文档是 MTPRO Engine module boundary 的压缩索引，保留目标模块、依赖方向、禁止路径和 MTP validation anchors。它不授权 live / broker / signed endpoint / real order runtime，不修改 Package.swift，不启动 Graphify / Figma / Symphony。

## Target Modules

`DomainModel`、`DataClient`、`DataEngine`、`MessageBus`、`Cache`、`Database`、`Trader`、`Trader/Strategies`、`Portfolio`、`RiskEngine`、`ExecutionEngine`、`ExecutionClient`、`Dashboard`、`Workbench`、`FutureLiveProConsole`。旧 `Core / Adapters / Persistence / Runtime / App / Dashboard / CSQLite` 只作为 migration source / compatibility shell。

## Source Layout

`Sources/DomainModel/`、`Sources/DataClient/<venue>/`、`Sources/DataEngine/`、`Sources/MessageBus/`、`Sources/Cache/`、`Sources/Database/`、`Sources/Trader/Strategies/<strategy>/`、`Sources/Trader/Coordination/`、`Sources/Portfolio/`、`Sources/RiskEngine/`、`Sources/ExecutionEngine/`、`Sources/ExecutionClient/`、`Sources/Dashboard/`。`Sources/Strategies/<strategy>/` 是 historical / compatibility anchor，不是 forward-looking canonical path。

## Dependency Direction

DataClient -> DomainModel；DataEngine -> DataClient / MessageBus / Cache；Trader -> DomainModel / MessageBus / Cache / Trader/Strategies / Portfolio / RiskEngine / ExecutionEngine；Dashboard / Workbench -> ReadModel / ViewModel exports。禁止 Strategies -> MessageBus -> ExecutionClient、Trader -> ExecutionClient、Workbench -> Database schema、DataClient -> signed endpoint、DataClient -> Cache -> account endpoint、FutureLiveProConsole -> current Workbench command surface。
